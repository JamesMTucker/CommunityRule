# TEI Transcriptions of Fragments relating to The Community Rule

## Description

The files here contain TEI transcriptions of the fragmentary manuscripts of the Qumran Community Rules. The TEI is a very stripped down version, yet includes the `regions of interest` for the corresponding IAA images.
